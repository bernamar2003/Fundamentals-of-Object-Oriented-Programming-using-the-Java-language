Resumo
Esta lição apresentou os conceitos fundamentais de classes e objetos em Java. Você aprendeu como definir classes como modelos para a criação de objetos, como criar objetos a partir de classes usando a newpalavra-chave `new`, e como acessar os atributos e métodos de um objeto usando o operador ponto (`.`). Também abordamos construtores, que são métodos especiais usados ​​para inicializar objetos, e a thispalavra-chave `new`, que é uma referência ao objeto atual. Compreender esses conceitos é crucial para dominar a programação orientada a objetos em Java.

Na próxima lição, exploraremos métodos e parâmetros com mais detalhes, incluindo sobrecarga de métodos, tipos de retorno e passagem de argumentos.
